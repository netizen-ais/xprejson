<x-pre-json> HTML Custom Element

<x-pre-json> is an HTML custom element that allows you to render JSON objects in HTML documents with human-readable formatting and expandable interaction for browsing deep JSON structures.

Usage

Include xprejson/xprejson.js in your HTML page and use the HTML custom element as follows: 

<script defer type="module" src="https://github.com/netizen-ais/xprejson/xprejson.js"></script>
<x-pre-json>
  {
    "hello": "world",
    "array": [1, 2, 3],
    "object": {
      "key": "value"
    }
  }
</x-pre-json>

Results into:

{ "hello": "world", "array": [1, 2, 3], "object": { "key": "value" } }
